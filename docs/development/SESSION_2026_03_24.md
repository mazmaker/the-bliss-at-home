### 1. LINE OA Quota Exceeded
- Free Plan จำกัด 200 ข้อความ/เดือน — ใช้ไปแล้ว ~572+
- **แก้ไข:** อัปเกรดเป็น Light Plan (200 บาท/เดือน) หรือรอ 1 เม.ย. quota reset

### 2. acceptJob() Profile ID vs Staff ID (อยู่ใน stash)
- `packages/supabase/src/jobs/jobService.ts` ส่ง profile_id แทน staff_id
- DB ที่ restore (17 มี.ค.) ทำงานได้ปกติ — แต่ถ้า DB เปลี่ยนอีกอาจเจอ bug นี้
- Fix อยู่ใน `git stash@{0}` — pop มาใช้เมื่อจำเป็น

### 3. DB Migration Drift (59% tables ไม่มี migration)
- รายละเอียดใน `docs/DB_MIGRATION_DIFF.md`
- ยังไม่ได้แก้ไข — เป็น technical debt

### 4. pnpm-lock.yaml ต้อง sync
- Vercel build จะ fail ด้วย `ERR_PNPM_OUTDATED_LOCKFILE` ถ้า lockfile ไม่ตรง
- **แก้ไข:** รัน `pnpm install --no-frozen-lockfile` แล้ว commit pnpm-lock.yaml ก่อน push

### 5. LIFF Endpoint URL
- ถ้าเคยเปลี่ยนเป็น tunnel URL ต้องเปลี่ยนกลับเป็น production: `https://staff.theblissmassageathome.com/staff/login`
