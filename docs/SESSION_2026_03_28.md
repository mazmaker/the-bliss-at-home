#### Bug Fixes ระหว่างพัฒนา
- **PDF download error** — `jsPDF.text(undefined)` → fallback `bill_number`
- **Hotel query error** — `contact_email` column ไม่มี → ใช้ `email` column แทน
- **[object Object] ใน email** — settings value เป็น JSONB `{value: "..."}` → extract `.value`
- **Email layout broken** — `display: flex` ไม่ทำงานใน email client → เปลี่ยนเป็น `<table>`
- **Bill number format ไม่สม่ำเสมอ** — `BILL-xxx` vs `INV-xxx` → ปรับทั้งหมดเป็น `INV-{YYYYMM}-{HOTEL8}-{RANDOM}`
